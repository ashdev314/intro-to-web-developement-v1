HTML Introduction

1. usage of <!DOCTYPE html> tag

<!DOCTYPE html> tells the browser to use HTML5 rules and render the page in standards mode. If we dont use doctype, the browser will render the page in quirks mode, which can lead to inconsistent behavior across different browsers and may not support modern HTML features.

Short History

Initially, DOCTYPE was written as a long SGML-based declaration with a public identifier and a DTD URL.

Example (older style): <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

Variations

In HTML 4.01, there were 3 common variations: Strict, Transitional, and Frameset.

In HTML5, this was simplified to one form: <!DOCTYPE html>.

2. lang attribute in html tag.

The lang attribute specifies the language of the element's content. It helps screen readers and search engines understand the language of the text.

Examples of lang attribute usage

1. Set language for the whole page (English):

<html lang="en">

2. Set regional English:

<html lang="en-US">

<html lang="en-GB">

3. Whole page in Hindi:

<html lang="hi">

Why this helps:

- Screen readers pronounce text correctly

- Search engines better understand page language

- Translation tools work more accurately

3. Emmet shortcuts in VS Code

Emmet is a powerful tool for quickly generating HTML structures.

Common Emmet shortcuts (mostly in HTML/CSS)

! -> HTML5 boilerplate

tag -> create tag (example: div)

tag.class -> class (example: div.card)

tag#id -> id (example: section#hero)

.class -> defaults to div (example: .container)

#id -> defaults to div (example: #app)

> -> child (example: ul>li)

+ -> sibling (example: h1+p)

^ -> go up one level (example: div>ul>li^p)

*n -> repeat (example: li*5)

$ -> numbering (example: li.item$*3)

{text} -> text content (example: a{Click here})

[attr=value] -> attributes (example: input[type=text])

() -> grouping (example: header>(nav>ul>li*3)+footer)

lorem or lorem20 -> dummy text

CSS Emmet examples

m10 -> margin: 10px;

p20 -> padding: 20px;

w100 -> width: 100px;

d:f -> display: flex;

jc:c -> justify-content: center;

ai:c -> align-items: center;

Useful ready-made examples

ul>li*3

table>tr*3>td*4

form>label+input+button

nav>ul>li>a
