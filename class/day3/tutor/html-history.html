HTML History

HTML (Hypertext Markup Language) was created by Tim Berners-Lee in late 1991 to help researchers at CERN share information. It has since evolved from a simple text-linking tool into a powerful standard for interactive web applications.

History & Key Milestones

  • Origin (1989 – 1991): Tim Berners-Lee proposed the World Wide Web and developed the first version of HTML with roughly 18 elements.
  • The Browser Wars (Mid-90s): Browsers like Netscape introduced proprietary tags (e.g., <blink>), leading to fragmentation. #
  • Standardization (1994-1997): The World Wide Web Consortium (W3C) was formed in 1994 to maintain web standards and resolve cross-browser inconsistencies.
  • Living Standard Transition: In 2019, the W3C and WHATWG agreed that HTML would no longer use traditional version numbers but would instead be a continuously updated "Living Standard".
