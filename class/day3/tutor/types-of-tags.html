img – An image. You use this to load images onto the page. This can be confusing because you can use CSS to bring in images too. attributes

Attributes allow you to modify behavior of an HTML tag. href only works on a tags. You can't add a href to a div and expect to suddenly work as a link. the type attribute on the input. By changing the type you're changing what sort of input is being shown. href and type are specific to a few tags. There are other attributes like class and id that are universal and can be applied to (nearly) all tags.
